## 函数命令

| 命令                   | 说明               |
| ---------------------- | ------------------ |
| package-list-packages  | 插件仓库插件列表   |
| package-install 插件名 | 安装插件           |
| package-delete 插件名  | 删除插件           |
| customize              | 系统变量列表       |
| eval-region            | 运行选中部分的代码 |





## spacemacs    VIM模式快捷键

| 快捷键        | 说明                                         |
| ------------- | -------------------------------------------- |
| SPC f e d     | 快速打开配置文件                             |
| SPC f e R     | 同步配置文件                                 |
| SPC q R       | 重启 emacs                                   |
| SPC h d       | 查看 describe 相关的文档                     |
| SPC h d f     | 查看指定函数的帮助文档                       |
| SPC h d b     | 查看指定快捷键绑定了什么命令                 |
| SPC h d v     | 查看指定变量的帮助文档                       |
| SPC f f       | 打开文件夹                                   |
| SPC /         | 用合适的搜索工具搜索内容                     |
| SPC s c       | 清除搜索高亮                                 |
| SPC f R       | 重命名当前文件                               |
| SPC b d       | 关闭当前 buffer                              |
| SPC f t       | 以侧边栏打开当前文件所有目录                 |
| SPC p t       | 打开当前文件所有的根目录                     |
| SPC 0         | 光标跳转到侧边栏                             |
| `SPC n`(数字) | 光标跳转到第 n 个 buffer 中                  |
| SPC w s       | 水平分割窗口                                 |
| SPC w v       | 垂直分割窗口                                 |
| SPC w d       | 关闭当前窗口                                 |
| SPC p p       | 切换项目                                     |
| SPC p D       | 在 dired 中打开项目根目录                    |
| SPC p f       | 在项目中搜索文件名                           |
| SPC p R       | 在项目中替换字符串                           |
| SPC j =       | 自动对齐，格式化代码                         |
| SPC '         | 打开/关闭 Shell                              |
| C-k           | 前一条 shell 命令，相当于在 shell 中按上箭头 |
| C-j           | 后一条 shell 命令，相当于在 shell 中按下箭头 |
| SPC t n       | 显示行号                                     |
| SPC h b       | 增加书签                                     |
| C-d           | 删除书签                                     |
| C-e           | 编辑书签                                     |
| C-o           | 在另一个窗口打开书签                         |
| /             | 在本文件搜索                                 |
| SPC /         | 在项目里智能搜索                             |
| SPC s b       | 在所有打开的buffer里搜索                     |
| SPC f r       | 最近打开的文件                               |
| SPC c l       | 注释行                                       |
| SPC c y       | 注释的同时并且复制相同的一份                 |
