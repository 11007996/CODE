## 颜色方案

- 次要背景: #F5F5F5
- 边框: #E0E0E0
- 文字: #212121
- 次要文字: #757575

### 深色主题
- 背景: #1E1E1E

- 次要文字: #B0B0B0

### 状态颜色
- 成功/运行中: #4CAF50 (绿色)
- 警告: #FF9800 (橙色)
- 错误: #F44336 (红色)
- 信息: #2196F3 (蓝色)
- 已停止: #9E9E9E (灰色)

## 交互效果

### 按钮
- 默认: 正常背景
- 悬停: 背景变亮 10%
- 按下: 背景变暗 10%
- 禁用: 透明度 50%

### 卡片
- 默认: 正常背景
- 悬停: 背景变亮，显示阴影
- 选中: 边框高亮

### 切换按钮
- 未选中: 灰色
- 选中: 强调色
- 悬停: 背景变化

---

## 响应式行为

### 窗口最小尺寸
- 最小宽度: 1200px
- 最小高度: 700px

### 面板宽度
- 左侧配置列表: 固定 320px
- 右侧状态面板: 固定 280px
- 中央预览区域: 自适应剩余空间

### 日志面板
- 折叠: 仅显示头部 (约 40px)
- 展开: 固定高度 250px
