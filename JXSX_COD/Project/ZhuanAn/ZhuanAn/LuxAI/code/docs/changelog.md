# Changelog

## v1.3.0

发布时间：2026-04-22

### 新增

- 算法插件化架构：将业务算法拆分为独立项目并编译为插件 DLL，支持按场景分发与更新，避免整包发布。
- S3 兼容对象存储同步能力：集成 S3/MinIO 同步服务，支持远程文件拉取与批量更新。
- Desktop 端“算法插件与资源更新”窗口：提供连接检查、配置管理、分类同步、统计与日志视图。
- Web 端远程同步入口与管理弹窗：对齐参考实现，支持配置、同步、状态与日志查看。

### 变更

- 插件与模型文件类型策略收敛：
  - 插件仅允许 `.dll`
  - 模型仅允许 `.onnx`
- 默认 S3 凭据调整为：
  - `AccessKey=admin`
  - `SecretKey=admin123`
- 配置加载增强：兼容旧版配置格式并可自动迁移到新结构。

### 工程结构

- 引入 `LuxVideoDet.Core/S3Storage` 模块及相关依赖。
- 调整插件目录策略，统一指向仓库根目录 `plugins/`（用于开发态与部署态一致性）。
- Web API 新增 S3 同步相关端点（配置、连接检查、分类同步、文件列表、下载、统计）。

### 说明

- 本版本重点解决“多产线节点插件与模型手工更新成本高”的问题，支持通过对象存储集中下发并在客户端一键同步。
