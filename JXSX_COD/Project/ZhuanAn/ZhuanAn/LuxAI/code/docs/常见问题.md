# 常见问题

## Q&A

### Q1：选用 CPU 能运行，GPU 会闪退，或报错找不到算法。

**A1：**  
常见原因是 NVIDIA 驱动版本不匹配，或 CUDNN 环境变量未正确配置。  
请确认：

- 已安装可用的 NVIDIA 驱动；
- CUDNN 版本为 **9.0 及以上**；
- CUDNN 相关路径已加入系统环境变量（并重启终端/系统后生效）。

---

### Q2：NG 回放录制功能使用不了。

**A2：**  
该功能依赖 FFmpeg。请先安装 FFmpeg，并在命令行执行以下命令验证：

```bash
ffmpeg --version
```

若命令有正常输出，即表示 FFmpeg 安装成功。

---

### Q3：Python 插件部署时，安装离线包失败。

**A3：**  
请先确认 Python 版本是否为 **3.11**。  
当前离线包与其他 Python 版本可能存在不兼容问题。

---

### Q4：YOLO 模型加载失败。

**A4：**  
当前模型加载仅支持 **YOLOv8 系列模型**。  
请确认模型版本与导出方式符合 YOLOv8 规范。

---

### Q5：Linux 部署时报 OpenCV 相关错误。

**A5：**  
大概率是系统依赖缺失。可先执行以下命令安装基础依赖：

```bash
sudo apt install -y libopencv-dev
```

安装后建议重新启动服务并再次验证。
